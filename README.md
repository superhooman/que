# que

Simple micro landing pages

Create effective micro landing page in several minutes without special skills

## Stack
- Next.js (Typescript)
- Prisma (MySQL)
- tRPC + zod

# Installation
1. Clone the repo
2. Create an `.env` file (more on that below)
3. Install dependencies `npm ci`
4. Start `npm run dev`

# .env
```
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
EMAIL_SERVER_USER=
EMAIL_SERVER_PASSWORD=
EMAIL_SERVER_HOST=
EMAIL_SERVER_PORT=
EMAIL_FROM=
DATABASE_URL=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
VERCEL_URL=
```
